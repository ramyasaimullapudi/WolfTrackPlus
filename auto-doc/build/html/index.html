Welcome to WolfTrackPlus’s documentation!

“WolfTrackPlus” is a program designed to keep track of the job applications that you wish to apply, already applied, and deadlines coming up for the job applications. It is a smart tracker that gathers general data and your custom information and helps job seekers keep track of their schedules.

Installation:

  1. Download the latest release from the repository based on your operating system. Releases

  2. Extract the zip file downloaded.

  3. Run “python3 main.py” from the extracted folder to launch the application.
